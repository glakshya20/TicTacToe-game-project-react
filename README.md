# ❌⭕ Tic Tac Toe Game (React)

🔗 **Live Demo:** https://playtictactoereact.netlify.app/

---

## 🚀 Overview

A fun and interactive **Tic Tac Toe game** built using React.  
Play against a friend, track moves, and enjoy a clean and responsive UI.

---

## ✨ Features

- 🎮 Two-player gameplay (X vs O)
- 🔄 Real-time turn switching
- 🏆 Winner detection logic
- ♻️ Restart game functionality
- ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack

- **Frontend:** React.js
- **Styling:** CSS
- **State Management:** React Hooks
- **Deployment:** Netlify

## ⚙️ Installation & Setup

```bash
git clone https://github.com/glakshya20/TicTacToe-game-project-react.git
cd TicTacToe-game-project-react
```

```bash
npm install
```

```bash
npm start
```

---

## 📦 Build for Production

```bash
npm run build
```

---

## 🌐 Deployment

Deployed using **Netlify**

- Build Command: npm run build  
- Publish Directory: build
